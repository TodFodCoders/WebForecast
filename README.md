# WebForecast
A simple website to display webforecast.

# Tech Stack
- ReactJS (Front End)
- TypeScript
- Node.js 
- Express.js (Back End)

# Features
- Real-time weather updates
- User-friendly interface
- Responsive design
- Integration with OpenMeteo API

# Installation
1. Clone the repository: `git clone https://github.com/todfodcoders/webforecast.git`
2. Navigate to the project directory: `cd webforecast`
3. Install dependencies: `npm install`
4. Start the development server: `npm start`
